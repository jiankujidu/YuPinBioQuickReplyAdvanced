# 优品生物快回复 - 高级版 Android应用

## 设计理念
- **现代化UI**：采用Material Design设计原则
- **用户体验**：流畅的动画和交互
- **功能丰富**：完整的悬浮窗、话术管理、数据库、权限系统
- **实用性强**：针对业务场景优化

## 核心功能

### 1. 悬浮窗系统（高级版）
- 精美悬浮窗UI
- 可拖拽、缩放
- 悬浮窗样式定制
- 快捷键触发悬浮窗

### 2. 话术管理系统（高级版）
- **三级分类**：公司/小组/私人话术
- **分类颜色**：多彩分类标签
- **标签系统**：多标签管理
- **批量操作**：批量导入、导出、编辑
- **搜索功能**：智能关键词搜索

### 3. 数据库系统
- SQLite + Room数据库
- 话术分类存储
- 用户权限存储
- 使用统计记录

### 4. 权限系统
- 管理员/小组长/普通用户权限
- 权限层级控制
- 话术访问权限控制

### 5. UI特色
- Material Design界面
- 动画过渡效果
- 主题颜色定制
- 响应式布局
